På indenskærs isrekognoscering onsdag den 23. juli 2014 mellem Bredefjord, Mågeløbene, Knækket, Thorstein Islænder, Qarmat, Skovfjord, Hollænderløbet, Julianehåbs Fjord, Mato Løbet, Narsaq Sund og Narsarsuaq blev følgende observeret:


10. I ruten inden om Thorstein Islænder: Var der enkelte isfjelde og skosser. Ud for Torssukatak var der spredte isfjelde og skosser.

11. I Torssukatak: Var der enkelte isfjelde og skosser.

12. I Knækket: Var der isfrit.

13. I Ikerasagssuaq: Var der enkelte skosser. Ud mod Sermilik var der enkelte isfjelde.

14. I Sermilik: Var der spredte isfjelde og skosser.

15. I Akugdleq: Var der enkelte isfjelde og skosser.

16. I Nordlige Mågeløb: Var der spredte isfjelde og skosser ved Qassimiut. Løbet var spærret af et isfjeld ved Pinguiarneq.

17. I Sydlige Mågeløb: Var der enkelte isfjelde og skosser. Ved Qornuata Nuna var der spredte isfjelde der kunne passeres.

18. I området vest for Ugarmiut: Var der spredte isfjelde og skosser.

19. I Qornoq: Var der spredte isfjelde der kunne passeres.

20. I Tunua: Var der enkelte isfjelde og skosser der kunne passeres.

21. I Ikerasak: Var der enkelte isfjelde og skosser der kunne passeres.

22. I Kuanit sava: Var der enkelte isfjelde og skosser der kunne passeres.

23. I Bredefjord: Var der spredte isfjelde og skosser, tiltagende til mange isfjelde og skosser ind mod Narsaq Sund.

24. I ruten Narssaq Sund - Tasiussaq: Var der mange isfjelde og skosser samt områder med 1-3/10 bræis.

25. I ruten nord om Qarmat til Hvide Humpel: Var der enkelte isfjelde og skosser.

26. I ruten syd om Qarmat: Var der enkelte isfjelde og skosser.

27. I ruten syd om Avatarmiut: Var der enkelte isfjelde og skosser.

28. I Hvide Humpel: Var der spredte isfjelde og skosser.

29. I Teisteløb: Var der enkelte skosser.

30. I Tuno: Var der spredte isfjelde og skosser.

31. I Tunuaraq: Var der spredte isfjelde og skosser.

32. I Narssaq Sund: Var der spredte isfjelde og skosser. 

33. I Narsaq havn: Var der enkelte isfjelde og skosser. Ved kajen lå der enkelte mindre skosser.

34. I Skovfjord: Var der enkelte isfjelde og skosser. Nær Narsaq Sund var der spredte isfjelde og skosser.

35. I Tunugdliarfik: Var der enkelte isfjelde og skosser. Ud for Qoroq var der mange isfjelde.

36. I ruten Hollænderløbet til Pardlit: Var der enkelte skosser. Ved Qatitit var der spredte isfjelde der kunne passeres.   

37. I Mato løb: Var der enkelte skosser.

38. Ved Mato Fyret: Var der isfrit.

39. I Qaqortoq havn: Var der isfrit.

40. I Julianehåbs Fjord: Var der enkelte isfjelde og skosser. Der var spredte isfjelde nær Pardlit.



Næste isrekognoscering forventes torsdag den 24. juli 2014.
