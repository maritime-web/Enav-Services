På indenskærs isrekognoscering torsdag den 10. juli 2014 mellem 
Prins Chr. Sund, Frederiksdal, Nanortalik, Julianehåbs Fjord, Hollænder Løbet, Mågeløbene, Knækket, Bredefjord, Narsaq Sund og Narsarsuaq blev følgende observeret:

12. I Knækket: Var der isfrit.

13. I Ikerasagssuaq: Var der enkelte isfjelde og skosser. 

14. I Sermilik: Var der ved mundingen enkelte isfjelde og spredte skosser. 

15. I Akugdleq: Var der enkelte isfjelde og skosser.

16. I Nordlige Mågeløb: Var der enkelte isfjelde og skosser.

17. I Sydlige Mågeløb: Var der enkelte isfjelde og skosser.

18. I området vest for Ugarmiut: Var der enkelte isfjelde og skosser.

19. I Qornoq: Var der enkelte isfjelde og spredte skosser.

20. I Tunua: Var der enkelte isfjelde og skosser.

21. I Ikerasak: Var der enkelte isfjelde og spredte skosser.

22. I Kuanit sava: Var der enkelte isfjelde og spredte skosser.

23. I Bredefjord: Var der fra udmundinge til midten af fjorden  spredte isfjelde og skosser. Fra midten af fjorden og til Narsaq Sund Mange isfjelde og skosser. 

24. I ruten Narssaq Sund - Tasiussaq: Var der mange isfjelde og skosser. I bunden af ruten var der 5-7/10 bræis.

25. I ruten nord om Qarmat til Hvide Humpel: Var der enkelte isfjelde og skosser.

26. I ruten syd om Qarmat: Var der enkelte isfjelde og skosser.

27. I ruten syd om Avatarmiut: Var der enkelte isfjelde og skosser.

28. I Hvide Humpel: Var der enkelte isfjelde og skosser.

30. I Tuno: Var der enkelte isfjelde og skosser.

31. I Tunuaraq: Var der  spredte isfjelde og skosser.

32. I Narssaq Sund: Var der mange isfjelde og skosser. 

33. I Narsaq havn: Var der isfrit ved kajen. 

34. I Skovfjord: Var der enkelte isfjelde og skosser.Ved udmundingen af Skovfjorden var der spredte isfjelde og skosser. 

35. I Tunugdliarfik: Var der ved Narsaq Sund enkelte isfjelde og skosser. Resten af fjorden var der spredte isfjelde og mange skosser.

36. I ruten Hollænderløbet til Pardlit: Var der enkelte isfjelde og skosser.

37. I Mato løb: Var der enkelte skosser.

38. Ved Mato Fyret: Var der enkelte isfjelde og skosser.

40. I Julianehåbs Fjord: Var der enkelte isfjelde og skosser.

41. I Igaliko Fjord: Var der enkelte isfjelde og skosser.

43. I I kerasarssungnut: Var der ved mundingen 5/10 brudt storis. Resten af løbet var der enkelte isfjelde og skosser.

44. I Ikerasarssuk kujatdleq: Var der spredte isfjelde og skosser.

47. Ved Paggivik Fyr: Var der enkelte isfjelde og skosser. 

48. I ruten Paggivik Fyr - Sardloq: Var der enkelte isfjelde og skosser.

49. I ruten nord og øst om Sardloq: Var der spredte isfjelde og mange skosser. 

50. Ikerasak ved Sardloq: Var der mange isfjelde og skosser. 

52. I Umanarssup tunua: Var der spredte isfjelde og mange skosser. 

53. I ruten syd om Imilik: Var der spredte isfjelde og mange skosser. 

54. I ruten Imilik til Qaersup ikerasa: Var der spredte isfjelde og mange skosser.

55. I Qaersup ikerasa: Var der spredte isfjelde og mange skosser, som kan passeres af mindre både. 

56. Nord om Qeqertarssuaq: Var der spredte isfjelde og mange skosser.

57. I nordlige Umanartut: Var der spredte isfjelde og skosser.

58. Syd om Qeqertarssuaq: Var der spredte isfjelde og mange skosser.

59. I Sydprøven Ikerasarssuk: Var der enkelte isfjelde og spredte skosser, som vil kunne passeres af mindre både.

60. I ruten syd om Sagdlia: Var der enkelte skosser. 

61. I ruten syd om Nunanguaq: Var der enkelte isfjelde og skosser. 

62. I sydlige Umanartut: Var der enkelte isfjelde og spredte skosser.

63. I ruten Lichtenau Fjord: Var der spredte isfjelde og skosser. 

64. I ruten Sydprøven - Unartoq: Var der spredte isfjelde og skosser.

65. I ruten Sydprøven-Sermersoq: Var der enkelte isfjelde og spredte skosser.

66. Nord for Arnarqat: Var der enkelte isfjelde og spredte skosser.

67. Syd for Arnarqat: Var der enkelte isfjelde og skosser.

68. I ruten Unartoq – Sermersoq: Var der spredte isfjelde og skosser. 

69. Nord for Sermersoq: Var der spredte isfjelde og skosser.

70. Nordøst for Sermersoq: Var der enkelte isfjelde og skosser.

71. I ruten Kap Egede – Nalunaq: Var der enkelte isfjelde og skosser. Ved udmundingen var der spredte isfjelde og skosser. 

72. Vest for Sermersoq: Var der enkelte isfjelde og spredte skosser.

73. Syd for Sermersoq: Var der spredte isfjelde og mange skosser. 

74. Øst for Thomsens Ø: Var der enkelte isfjelde og skosser.

75. I Nanortalik Ikerasarssuk: Var der enkelte isfjelde og skosser.

76. I Quagssup tunua: Var der spredte isfjelde og skosser. 

77. I Nanortalik havn: Var der isfrit.

78. I ruterne til Nanortalik omkring Ortusugssuk: Var derspredte isfjelde og skosser. 

79. Ved Nanortalik i nordvest - sydøst fyrlinien: Var der enkelte isfjelde og spredte skosser.

80. Ved Nanortalik i nord - syd fyrlinien: Var der spredte isfjelde og skosser. 

81. Ved Nanortalik i øst - vest fyrlinien: Var der spredte isfjelde og mange skosser. 

83. I båkeruten Nanortalik - Frederiksdal: Var der spredte isfjelde og mange skosser. 

84. I ruten Nanortalik - Frederiksdal: Var der i den østlige del spredte isfjelde og mange skosser. I den Vestlige del var der enkelte isfjelde og spredte skosser. 

85. I ruten Frederiksdal - Augpilagtoq: Var der spredte isfjelde og mange skosser. 

86. I ruten Augpilagtoq – Sermerunerit: Var der spredte isfjelde og mange skosser.

87. I Prins Christian Sund ved det smalle sted: Var der enkelte isfjelde og skosser.

88. I Prins Christian Sund østlige del: Var der enkelte isfjelde og spredte skosser.

89. I Prins Christian Sund Ø-lig ved mundingen: Var der mange isfjelde og skosser. Ud for mundingen af Prins Chr. Sund var der revler med 9/10 storis. 



Næste isrekognoscering forventes onsdag den 16. juli 2014.
